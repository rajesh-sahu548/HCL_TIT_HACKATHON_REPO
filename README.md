# Selenium Automation Framework

## Project Overview
This project is a Selenium Automation Framework developed using **Java, Selenium WebDriver, TestNG, and Maven**.  
It is designed to automate web application testing with a **Page Object Model (POM)** structure and support for **data-driven testing, reporting, and parallel execution**.

The framework helps testers create maintainable and reusable automation scripts.

---

## Technologies Used

- Java
- Selenium WebDriver
- TestNG
- Maven
- Git & GitHub
- Page Object Model (POM)

---

## Framework Features

- Page Object Model design pattern
- Data-driven testing using Excel / DataProvider
- Screenshot capture on test failure
- TestNG reports
- Configurable browser setup
- Reusable utilities
- Parallel test execution
